Global Attributes with examples

  1. accesskey – Keyboard shortcut to focus an element
    <button accesskey="s">Save</button>
  2. autocapitalize – Controls automatic text capitalization
    <input type="text" autocapitalize="words">
  3. autofocus – Automatically focuses an element on page load
    <input type="text" autofocus>
  4. class – Assigns class names for styling or scripting
    <div class="card highlight"></div>
  5. contenteditable – Makes content editable by the user
    <p contenteditable="true">Edit me</p>
  6. data-* – Stores custom data
    <div data-user-id="123"></div>
  7. dir – Sets text direction
    <p dir="rtl">مرحبا</p>
  8. draggable – Makes an element draggable
    <img src="img.png" draggable="true">
  9. enterkeyhint – Changes Enter key label on mobile keyboards
    <input enterkeyhint="search">
  10. hidden – Hides the element
    <div hidden>Hidden content</div>
  11. id – Unique identifier
    <section id="main"></section>
  12. inert – Disables interaction inside an element
    <div inert><button>Click</button></div>
  13. inputmode – Suggests keyboard type
    <input inputmode="numeric">
  14. is – Extends a built-in HTML element
    <button is="fancy-button">OK</button>
  15. itemid – Unique ID for microdata
    <div itemid="https://example.com/item/1"></div>
  16. itemprop – Property of a microdata item
    <span itemprop="name">Book</span>
  17. itemref – References related microdata elements
    <div itemref="details"></div>
  18. itemscope – Creates a microdata item
    <div itemscope></div>
  19. itemtype – Defines microdata vocabulary
    <div itemtype="https://schema.org/Person"></div>
  20. lang – Specifies language
    <p lang="en">Hello</p>
  21. nonce – Allows trusted scripts/styles under CSP
    <script nonce="abc123"></script>
  22. popover – Enables popover behavior
    <div popover>Popover content</div>
  23. role – Accessibility role
    <nav role="navigation"></nav>
  24. slot – Assigns content to a Web Component slot
    <span slot="title">Header</span>
  25. spellcheck – Enables/disables spell checking
    <textarea spellcheck="true"></textarea>
  26. style – Inline CSS styling
    <p style="color:red">Text</p>
  27. tabindex – Controls tab order
    <div tabindex="0">Focusable</div>
  28. title – Tooltip text
    <button title="Click me">Hover</button>
  29. translate – Controls translation
    <p translate="no">BrandName</p>
